GMate

This package contains some gedit improvements to make it more similar to TextMate. The package contain code snippets, plugins, and an automatic registration of rails-related files.

If you have issues with one of the plugins included in Gmate or have suggestions please fill an issue in http://github.com/gmate/gmate/issues

This Fork

This is a fork of the Gmate project to port the remaining plugins to Gedit3. While the master branch is stable, all other branches are experimental and not meant for production environments.

If you want to download the most stable and up-to-date version of Gmate for everyday use, visit the main Gmate repo at http://github.com/gmate/gmate.
